 CGI - Common Gateway Interface - интерфейс общего шлюза
 Протокол, согласно которому веб-сервер может запускать сторонние программы и передавать их управление для формирования ответа

            HTTP                         разбор запроса
 browser ---------->    :80 <---> Apache --------------> Python
         <----------        <-----       <-------------- index.py print(...)

Веб сервер (Apache) получает запрос (HTTP-пакет), разбирает его (заголовки, тело, параметры) создает из них переменные и передает
 их в скрипт. Скрипт Python выводит данные (print), этот вывод "перехватывается" веб сервером и формирует пакет-ответа

1. Настройка Apache
    - устанавливает, проверяем на возможность запуска
    - настраиваем локальный (виртуальный) хостинг
      = открываем папку с Apache [C:\xampp\apache]
      = заходим в папку [conf], если есть, то [extra] открываем конфигурацию а) httpd-vhosts.conf или б) httpd.conf
      = создаем настройки для нашего сайта (хоста)
